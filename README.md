# Python版QQ群发器（TIM版）

## 本项目只为学习只用，请勿用于任何违反国家法律法规的用途。

## 系统要求
1. Windows 7
2. Python 2.7.14
3. TIM 1.2.x

## 功能特色
1. 自动登录，识别验证码
2. 自动识别登录结果：密码错误、账号冻结、保护模式、登陆超时
3. 自动识别群组，支持自定义群发组名
4. 支持文字+随机表情+图片发送

## 环境搭建
1. Python 环境搭建，安装依赖
2. 注册大漠插件到系统
3. 使用自己的QQ账号和超人打码账号替换代码中的账号位置
4. 执行群发：python app.py
